JavaScript

JavaScript é uma linguagem de programação utilizada para o lado do cliente em páginas web, sendo também usado para o lado do servidor, com a utilização do Node.js. Também é possivel utilizá-lo para aplicações mobile, basta usar o framework React Native.

Origem

JavaScript criada por Brendan Eich em 1995. Foi criada para o navegador Netscape Navigator 2 com o objetivo de ser uma linguagem de script para o uso principal da empresa. Logo após seu lançamento, muitos navegadores começaram a dar suporte a JavaScript. No começo, sofreram vários problemas de segurança, mas não tinha outra solução a não ser usar JS. Em 2005, houve uma virada de chave com a criação do Chrome V8. Isso proporcionou um alto desempenho. Logo depois, Ryan Dahl criou o Node.js. Fornencendo a possibilidade de executar seu código para o lado do servidor.
